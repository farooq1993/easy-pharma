How to generate App Password?

Google requires a separate 16-digit App Password to let external applications connect securely via IMAP. Follow these steps:

  1. Turn on 2-Step Verification
    Go to your Google Account -> Security. Ensure "2-Step Verification" is ON.
  2. Open App Passwords Settings
    Search for "App Passwords" in the top search bar of your Google Account page, or find it at the bottom of the "2-Step Verification" page.
  3. Create "Easy Pharma" password
    Select App as "Other (Custom name)", type "Easy Pharma", and click Generate.
  4. Copy the Code
    Google will display a yellow box containing a 16-letter code (e.g., `abcd efgh ijkl mnop`). Copy it (without spaces) and paste it into the field on the left.
